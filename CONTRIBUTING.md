# Contributing to runstat

We welcome contributions to runstat! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/runstat.git`
3. Create a feature branch: `git checkout -b feature-name`

## Development

### Prerequisites
- macOS 10.13 or later
- Xcode with Swift support

### Building
```bash
swift build -c release
```

### Testing
Test the app on different macOS versions and system loads.

## Submitting Changes

1. Make your changes
2. Test thoroughly
3. Commit with clear messages
4. Push to your fork
5. Create a Pull Request

## Code Style

- Follow Swift naming conventions
- Add comments for complex logic
- Keep functions focused and small

## Reporting Issues

- Use GitHub Issues
- Include macOS version
- Describe expected vs actual behavior
- Provide system information if relevant

Thank you for contributing!
